**Yelp Review Full Star Dataset**

**Summary**

The Yelp reviews dataset consists of reviews from Yelp. It is extracted from the Yelp Dataset Challenge 2015 data.


**Basic stats:**
+ \# items = 700,000
+ \# label = 5

**Basic Unit**: paragraph

** Split: train : test = 650,000 : 50,000 **
