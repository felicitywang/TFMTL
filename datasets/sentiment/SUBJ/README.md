**Subjectivity Dataset**

**Summary**

Automatically labeled movie review snippets. All the Rotten Tomatoes movie reviews are labeled subjective, and all IMDB objective.

**Basic stats:**

+ \# items = 10,000
+ \# labels = 2
    - 0: subjective
    - 1: objective

**Basic Unit**: sentence

**Split**: none
