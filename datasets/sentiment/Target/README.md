**Target-dependent Sentiment Analysis**

**Summary**

Seeks to classify the sentiment of a text's author towards an entity that occurs in the text as positive, negative, or neutral.

**Basic stats:**

+ \# items = 6,248
+ \# labels = 3
    - 0: negative
    - 1: neutral
    - 2: positive

**Basic Unit**: sentence

**Split**: train/test
