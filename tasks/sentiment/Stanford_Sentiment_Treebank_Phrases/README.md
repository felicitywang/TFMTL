**Stanford Sentiment Treebank - Phrases**

**Summary**

Phrases from Rotten Tomatoes reviews with sentiment labels.


**Basic stats:**

+ \# items = 239,231
+ labels : a real-valued positivity probability in [0,1], can be mapped into 5 categories:
    - [0, 0.2]: very negative
    - (0.2, 0.4]: negative
    - (0.4, 0.6]: neutral
    - (0.6, 0.8]: positive
    - (0.8, 1.0]: very positive

**Basic Unit**: phrases
