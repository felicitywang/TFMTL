**IMDb Movie Review Dataset**

**Summary**

IMDb movie reviews.

**Basic stats:**

+ \# items = 50,000(train 25,000, test 25,000) (unlabeled: 50,000)

+ \# labels = 2
    - -1: neg
    - +1: pos

**Basic Unit**: document

**Split**: train : test = 1 : 1
