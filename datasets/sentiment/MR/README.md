**Movie Review Dataset**

**Summary**

Rotten Tomatoes review sentences with polarity labels. Sentence polarity dataset v1.0.

**Basic stats:**

+ \# items = 10,662
+ \# labels = 2
    - 0: negative
    - 1: positive

**Basic Unit**: sentence

**Split**: not given

**bibtex**
